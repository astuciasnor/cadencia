# Cadencia

Aplicativo web de foco, pausas e progresso diario.

## Publicacao no GitHub Pages

1. Crie um repositorio vazio chamado `cadencia` na conta `astuciasnor`.
2. Envie este projeto para a branch `main`.
3. No GitHub, abra `Settings > Pages`.
4. Em `Build and deployment`, escolha `Deploy from a branch`.
5. Selecione a branch `main` e a pasta `/(root)`.

Depois disso, o app deve ficar disponivel em:

`https://astuciasnor.github.io/cadencia/`
